准备工作

  1. 下载并安装node.js

    https://nodejs.org/en/download/

  2. 下载并安装git

    https://git-scm.com/

  3. 检查
    在开始菜单里打开git bash, 输入下行命令查看node.js版本

    node -v

输入下行命令查看npm(包管理器)版本

npm -v

换源

由于国内访问外网速度感人, 先换国内源
在刚打开的git bash里键入下行命令

npm install -g cnpm –registry=https://registry.npm.taobao.org

安装及配置

键入命令 安装hexo博客框架

cnpm install -g hexo-cli

验证是否安装成功

hexo -v

建立一个文件夹, 在新建文件夹里右键, 打开git bash, 执行命令

hexo init

这样就算成功了

启动博客

hexo s


打开http://localhost:4000就可以看到

使用ctrl+C断开连接

新建一个博客

hexo n “name”


之后就可以使用编辑器对这个文件进行编辑了
编辑结束后, 保存并退出
在git bash下进入你的blog文件夹, 进行清除和生成操作

hexo c
hexo g

再次启动hexo

hexo s

就能在localhost:4000端口看见新建的博客了

断掉链接就好了

ctrl+c

将博客部署到远端

GitHub上新建一个项目, 要求: 箭头里的两个字符串必须一致(笔者之前搞过一次, 所以名字重复就给我报错了)

之后就什么不用管, 直接创建就好了.

接着用git bash进行预处理, 在您的博客目录下安装工具

cnpm install –save hexo-deployer-git
npm install hexo-deployer-git –save

对blog/_config.yml进行编辑, 按照下图进行操作(网址该为自己刚创建的项目网址)

将hexo部署到远端

hexo d
第一次操作的同学, 在这部命令之前要先进行身份编辑, 先进行下面的命令
git config –global user.name” “ –>引号里面的就是你的GitHub名字
git config –global user.email “ “ –>里面填写你的注册邮箱
最后执行 hexo d

跳出下面界面

请同学坦诚的设置下, 要是跳出SSH的对话框要求填写GitHub用户名, 密码等等也请坦诚的设置下.
成图如下:

刷新项目页

然后就可以用浏览器打开您的博客了, 以上图为例: 我的项目名是ranself.github.io, 那么就只需要打开https://ranself.github.io就可以了.

这里建议用Google chrome或新edge打开, 若出现这个错误

DNS_PROBE_FINISHED_NXDOMAIN

建议换个浏览器, 或是设备进行登陆测试, 例如笔者就是用手机夸克浏览器登陆测试的.
若其他设备或浏览器可以登录则需要更改您电脑的DNS, 方法如下

https://www.wbolt.com/how-to-easily-flush-dns-cache.html

之后再打开您的博客尝试下
